# 路笛诗斋 v1.1\n\n诗词收录 · AI 赏析 · Word 导出 · JSON 备份\n\n## 安装到手机\n\n1. 将 `index.html`、`sw.js`、`manifest.json` 三个文件上传到 GitHub Pages\n2. 用手机浏览器打开对应网址\n3. 浏览器会自动提示"添加到主屏幕"（或手动在菜单中选）\n4. 安装后像普通 App 一样使用，全屏、可离线\n\n## 配置 AI 接口\n\n1. 打开 App → 设置\n2. 去 [platform.deepseek.com](https://platform.deepseek.com) 注册，获取免费 API Key\n3. 填入 API Key → 点击「测试连接」\n4. 也可换成任何兼容 OpenAI 格式的 API\n\n## 功能\n\n- 📝 **录入**：输入诗词 → AI 自动拟题 + 赏析 → 保存到本机\n- 📚 **诗集**：浏览、搜索、编辑、删除已有诗词\n- 📄 **导出 Word**：单首或全部，格式为：标题居中→诗词居中→【作品赏析】左对齐→赏析两端对齐，每首自动分页\n- 📤 **备份 JSON**：所有诗词导出为 JSON 文件\n- 📥 **导入 JSON**：从 JSON 文件批量导入，自动跳过重复\n- 🔒 **隐私安全**：所有数据保存在手机本地，不经过任何服务器\n\n## 微信 vs 浏览器数据同步\n\n微信内置浏览器和系统浏览器是相互隔离的，本地数据天然不互通。\n同步方法：在一个入口中点击「备份 JSON」，另一个入口中点击「导入 JSON」即可。\n\n## 费用\n\n- GitHub Pages 部署：免费\n- DeepSeek API：注册送免费额度，个人使用几乎不花钱\n- 数据存储：全部在手机本地，零云存储费用
